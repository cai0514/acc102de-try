# S&P 500 Consumer Staples Monthly Volatility Tool

## 1. Purpose
This interactive tool supports analyzing monthly return and volatility patterns for the S&P 500 Consumer Staples sector.

## 2. Data
- Source: WRDS-CRSP
- Period: 2021-2025
- Frequency: Monthly

## 3. How to Run Locally
pip install -r requirements.txt
streamlit run app.py

## 4. Limitations
Uses simulated monthly data for demonstration. Real data can be imported from WRDS-CRSP in CSV format.
